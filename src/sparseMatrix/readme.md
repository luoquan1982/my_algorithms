# 稀疏数组
## 基本介绍(举例说明)
稀疏矩阵如下
```
0   0   0   22  0   0   5
0   11  0   0   0   17  0
0   0   0   -6  0   0   0
0   0   0   0   0   39  0
9   0   0   0   0   0   0
0   0   28  0   0   0   0
```
转换
 
|  | 行 | 列 | 值 |
| -------- | -------- | -------- | -------- |
| [0] | 6 | 7 | 8 |
| [1] | 0 | 3 | 22 |
| [2] | 0 | 6 | 15 |
| [3] | 1 | 1 | 11 |
| [4] | 1 | 5 | 17 |
| [5] | 2 | 3 | -6 |
| [6] | 3 | 5 | 39 |
| [7] | 4 | 0 | 91 |
| [8] | 5 | 2 | 28 |
- 稀疏数组的处理方法  
1）记录数组一共有几行几列，有多少个不同的值  
2）把具有不同值的元素的行列及值记录在一个小规模的数组中，从而缩小程序的规模
